# F1-Analytics
## Overview
Streamlit UI to visualise F1 data for drivers and races
## Set Up and Run
### Using Windows .bat file
Run Setup only on the first time
```powershell
setup
```
```powershell
run
```
### Manually Install Dependencies and Run
```powershell
python -m venv venv
pip install -r requirements.txt
```
```powershell
streamlit run app.py
```
